# 步骤图片功能

## 功能说明

现在每个烹饪步骤都可以添加图片了！在编辑菜谱时，每个步骤下方都有图片上传功能。

## 主要更改

### 1. 数据结构
步骤从简单的文本字符串改为对象：
```javascript
// 之前
steps: [{ text: '步骤描述' }]

// 现在
steps: [{ text: '步骤描述', image: '图片URL' }]
```

### 2. 编辑页面 (RecipeEdit.vue)
- ✅ 每个步骤卡片新增图片上传区域
- ✅ 支持上传、预览、更换和移除图片
- ✅ 上传时显示进度动画
- ✅ 图片限制：jpg/png/webp/gif，最大 5MB
- ✅ 新增 `uploadingSteps` 状态管理每个步骤的上传状态
- ✅ 使用 `stepFileInputs` ref 数组管理多个文件输入

### 3. 详情页面 (RecipeDetail.vue)
- ✅ 步骤区域显示步骤图片（如果有）
- ✅ 图片样式：圆角、阴影、响应式
- ✅ 兼容旧数据格式（纯文本步骤）

### 4. 后端 API
- ✅ 无需修改，已支持 JSON 格式的 steps 数组
- ✅ 图片上传使用现有的 `/api/recipes/upload` 接口

## 使用方法

1. 进入菜谱编辑页面
2. 在"烹饪步骤"区域，每个步骤下方有"添加步骤图片"按钮
3. 点击按钮选择图片上传
4. 上传成功后可以预览、更换或移除图片
5. 保存菜谱后，图片会在详情页显示

## 技术细节

- 图片上传复用封面图的上传逻辑
- 每个步骤独立管理上传状态
- 支持移动端触摸操作
- 响应式设计，适配各种屏幕尺寸
